# Documentação Técnica do GreenChain

Bem-vindo à documentação oficial do **GreenChain** (Recycle-to-Earn). A documentação foi separada em módulos específicos para facilitar a leitura e manutenção à medida que o projeto cresce.

## Módulos da Documentação

1. **[Módulo do Backend (API & Sessões)](file:///c:/Users/joaof/Downloads/projetos/greenchain/docs/backend.md):**
   * Detalha os Route Handlers, autenticação de sessão do NextAuth, validação de schemas com Zod, integração Prisma ORM e controle de nonce/OTP via Redis.
2. **[Módulo do Frontend (Interface & Web3)](file:///c:/Users/joaof/Downloads/projetos/greenchain/docs/frontend.md):**
   * Detalha o framework visual Next.js + React 19, customização visual com Tailwind CSS v4 e a camada Web3 (Wagmi, Viem, RainbowKit e React Query).
3. **[Módulo do Contrato Inteligente (Solidity na Celo)](file:///c:/Users/joaof/Downloads/projetos/greenchain/docs/contract.md):**
   * Detalha as escolhas de rede (Celo), modificadores de acessos e a lógica conceitual de validação e disparo das recompensas.
